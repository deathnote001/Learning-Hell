# 每日学习平台 · 部署指南

## 第一步：配置 Supabase 数据库

1. 登录 https://supabase.com，进入你的项目
2. 左侧菜单点 **SQL Editor**
3. 点 **New query**，把 `supabase_schema.sql` 文件内容全部粘贴进去，点 **Run**
4. 看到 "Success" 即建表完成

### 设置两个账号密码

在 SQL Editor 运行以下命令（把 `你的密码` 替换成真实密码）：

```sql
-- 先在浏览器控制台生成密码hash：
-- 打开任意网页，按F12，在Console粘贴以下代码：
-- async function h(pw){const b=await crypto.subtle.digest("SHA-256",new TextEncoder().encode(pw));return Array.from(new Uint8Array(b)).map(x=>x.toString(16).padStart(2,"0")).join("");}
-- 然后运行：h("你的密码").then(console.log)  // 复制输出的hash

UPDATE users SET password_hash = '这里填学生账号的hash' WHERE username = 'student';
UPDATE users SET password_hash = '这里填家长账号的hash' WHERE username = 'parent';
```

> 💡 **简单方法**：打开浏览器 → 按 F12 → Console 标签 → 粘贴上面的函数定义 → 然后输入 `h("你的密码").then(console.log)` → 复制输出结果

### 获取 Supabase 连接信息

左侧菜单 **Project Settings** → **API**：
- **Project URL**：形如 `https://xxxxx.supabase.co`
- **anon public key**：一串很长的字符串

把这两个值记下来，下一步用。

## 第二步：上传代码到 GitHub

1. 登录 GitHub，点右上角 **+** → **New repository**
2. 仓库名填 `learning-app`，选 **Private**（私人仓库），点 **Create repository**
3. 在你的电脑上，把 `learning-app` 文件夹上传：

**方法A（命令行，推荐）**：
```bash
cd learning-app
git init
git add .
git commit -m "init"
git branch -M main
git remote add origin https://github.com/你的用户名/learning-app.git
git push -u origin main
```

**方法B（网页上传）**：
在 GitHub 新建的仓库页面，点 **uploading an existing file**，把所有文件拖进去（注意保持目录结构），提交即可。

---

## 第三步：部署到 Vercel

1. 登录 https://vercel.com（可以用 GitHub 账号登录）
2. 点 **Add New Project** → **Import Git Repository**
3. 找到 `learning-app`，点 **Import**
4. 在 **Environment Variables** 部分，添加两个变量：
   - `VITE_SUPABASE_URL` = `https://xxxxx.supabase.co`（你的 Project URL）
   - `VITE_SUPABASE_ANON_KEY` = `你的 anon public key`
5. 点 **Deploy**，等待1-2分钟
6. 部署成功后，Vercel 会给你一个网址，例如 `https://learning-app-xxx.vercel.app`

---

## 第四步：在 iPad 上使用

1. 用 Safari 打开 Vercel 给的网址
2. 点底部分享按钮 → **添加到主屏幕**
3. 以后直接从主屏幕图标打开，像 App 一样用

**学生账号**：username = `student`，密码 = 你设置的  
**家长账号**：username = `parent`，密码 = 你设置的

## 更换题库（后续维护）

登录家长账号 → **内容管理** 标签 → 选择对应模块上传文件

### 单词库格式（TXT，每行一个单词）
```
water	水
fire	火
stone	石头
```
用 Tab 键分隔单词和中文意思。

### 单词库格式（JSON）
```json
[
  {"word": "water", "meaning": "水"},
  {"word": "fire", "meaning": "火"}
]
```

### 古诗词库格式（JSON）
```json
[
  {
    "id": "p001",
    "title": "观沧海",
    "author": "曹操",
    "dynasty": "东汉",
    "grade": "初一上",
    "lines": [
      "东临碣石，以观沧海。",
      "水何澹澹，山岛竦峙。",
      "树木丛生，百草丰茂。"
    ]
  }
]
```

### 历史题库格式（JSON）
```json
[
  {
    "id": "h001",
    "grade": "初一上",
    "topic": "夏商周",
    "stem": "西周实行分封制的根本目的是（   ）",
    "options": ["A. 发展农业生产", "B. 巩固西周统治", "C. 奖励功臣", "D. 扩大疆域"],
    "correct": 1,
    "explain": "分封制根本目的是巩固西周统治。"
  }
]
```
> `correct` 是正确答案的下标，从 0 开始（A=0, B=1, C=2, D=3）

## 常见问题

**Q：两台 iPad 数据不同步？**  
A：确认两台设备登录的是同一个账号（都用 `student`），数据存在 Supabase 云端，任何设备登录同账号都能看到。

**Q：发音没有声音？**  
A：确认 iPad 没有静音，Safari 允许网页播放音频。第一次可能需要先点一下屏幕才能播放。

**Q：想改密码怎么办？**  
A：在 Supabase SQL Editor 运行：
```sql
-- 先用上面的方法生成新密码的hash，然后：
UPDATE users SET password_hash = '新hash' WHERE username = 'student';
```

**Q：Vercel 网址太难记？**  
A：在 Vercel 项目设置里可以绑定自定义域名（需要购买域名），或者直接保存到主屏幕就行。
